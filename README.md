### PROGRAM:
```
#Program to find the distance between two points.
#Developed by: Yamuna M
#RegisterNumber:23005405
import math
a=[10,6]
b=[4,2]
distance=math.sqrt(((b[0]-a[0])**2)+((b[1]-a[1])**2))
print("{:.2f}".format(distance))
```  
